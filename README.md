# Official Leyu Lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为页面资源的集中管理与分发渠道。

## 项目简介

本仓库旨在为各类独立 HTML 页面提供统一的归档与展示空间。所有页面均为静态资源，可独立运行，不依赖后端服务。适用于文档展示、工具页面、信息卡片、实验性页面等多种场景。

## 目录结构说明

```
├── README.md          # 本说明文件
├── pages/             # 存放所有独立 HTML 页面
│   ├── page-1/       # 示例页面目录
│   ├── page-2/       # 示例页面目录
│   └── ...
└── assets/            # 公共资源（如 CSS、JS、图片等，可选）
```

- `pages/`：每个子目录对应一个独立页面，包含该页面的所有 HTML、样式及脚本文件。
- `assets/`：存放多个页面可能共用的资源文件，便于统一维护。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开运行。
- 每个页面拥有独立的目录，避免资源冲突。
- 页面内容可能包含实验性、临时性或长期维护的版本，具体状态请参见目录内的说明文件（如有）。
- 本仓库不承诺对任何页面进行持续更新或长期保留，请以实际提交记录为准。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立页面，或对现有页面进行改进。
- 提交时请确保页面为完整可用的静态 HTML，且不包含外部敏感链接或恶意内容。
- 请保持目录结构清晰，每个页面单独一个文件夹，并附带简要说明（可选）。
- 本仓库不接受任何形式的商业推广、广告内容或诱导点击的页面。

## 使用方式

可通过以下方式访问本仓库中的页面：

1. 直接克隆或下载仓库，在本地打开 HTML 文件。
2. 通过 GitHub Pages 部署（需自行配置），即可在线访问。
